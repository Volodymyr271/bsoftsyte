Huckster

Программа для управления товарным учетом, инвентаризацией и заказами в розничной и оптовой торговле.

Описание

Описание программы Huckster

Область применения

  • Магазин.
  • Торговая точка.
  • Производство.
  • Торговая сеть.

Основные возможности

  • Суммарно-денежный учет товара
  • Множество торговых точек
  • Заказ товара для торговли
  • Инвентаризация
  • Формирование документов для бухгалтерской отчетности
  • Встроенная система безопасности
  • Встроенная система помощи

Интеграция

  • Storage — программа для складского учета, производства и торговли
  • PayOff — программа для ведения бухгалтерского учета
  • Prices — программа для формирования заказов на закупку

Технические детали

  • Требуется регистрация (бесплатная, коммерческая или сетевая), иначе ввод документов ограничен.
