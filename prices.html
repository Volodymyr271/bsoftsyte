Prices

Работа с прайс-листами поставщиков

Описание программы Prices

Область применения

  • Оптовая и розничная торговля.

Основные возможности

  • Формирование заказов на закупку товаров по прайс-листам поставщиков:
    • Импорт прайс-листов и перечня товара в формате Microsoft Excel.
    • Автоматизированный выбор оптимальной закупочной цены.
    • Экспорт заказов в формате Microsoft Excel для отправки поставщику.
  • Встроенная система безопасности:
    • Регистрация пользователей и работа с документами под учетной записью.
    • Резервное копирование базы данных.
  • Встроенная система помощи:
    • Контекстная справка в формате Microsoft HTML Help.

Интеграция

  • Storage — складской учет, производство, торговля.
  • Huckster — управление магазинами и торговыми точками.

Особенности

  • Простота и удобство использования для формирования заказов.
  • Возможность легкого поиска и наценки товаров в прайс-листах.
